# Amazon-clone
Built a responsive Amazon Clone using HTML, CSS, and JavaScript. The project replicates Amazon's user interface with a navigation bar, search bar, product listings, categories, shopping cart, and responsive layout. It showcases front-end development skills, clean UI design, interactive components, and mobile-friendly user experience.

## Features
- Responsive Design
- Navigation Bar
- Product Cards
- Search Bar UI
- Hero Banner
- Footer

## Technologies Used
- HTML5
- CSS3
- JavaScript
